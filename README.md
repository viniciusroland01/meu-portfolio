# Portfolio | Vinícius Roland
> Backend Developer Student (ADS)

Este repositório contém meu portfólio profissional desenvolvido com **Django**. O projeto foi construído para ser o hub central das minhas automações e aplicações web.

## 🛠 Tech Stack
- **Core:** Python & Django (MVT)
- **Database:** SQLite (Persistência de projetos e perfil)
- **Frontend:** Bootstrap 5 & Font Awesome

## 📁 Destaques do Projeto
- **CRUD Administrativo:** Gerenciamento total via Django Admin.
- **Integração Social:** Seção de contatos otimizada (WhatsApp/LinkedIn/GitHub).
- **Seção de Skills:** Organizada por ícones dinâmicos de tecnologia.
- **Responsive Design:** Adaptado para mobile e desktop.

## 🚀 Como subir o ambiente
1. Ativar ambiente virtual: `.\venv\Scripts\activate`
2. Instalar dependências (caso existam): `pip install django`
3. Executar: `python manage.py runserver`
